A Full stack developer is a person who has ultimate knowledge of both FrontEnd 
development and BackEnd development, He/She can also develop and manipulate Client 
side (FrontEnd development) and Server side (BackEnd development).
     TOOLS AND LANGUAGES FULL STACK DEVELOPER USE
Tools: Visual Studio Code, Github, Slack, Backbone etc...
Language used for Client side (FrontEnd): HTML, CSS, JavaScript, Bootstrap etc...
Language used for Server side (BackEnd): SQL, PHP, Node.js, Python, C++ etc...
